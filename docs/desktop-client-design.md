# 桌面客户端 - 产品设计文档

## 定位

类似 iTerm 的桌面终端客户端，专为使用 Claude Code / OpenCode 优化，支持 macOS / Windows / Linux 三平台。

---

## 技术栈

| 层 | 技术 |
|----|------|
| 后端 | Tauri 2 (Rust) |
| 前端 | Vue 3 + Vite + xterm.js |
| 包管理 | pnpm |
| 通信 | Tauri IPC（不走 WebSocket） |
| PTY | 本地直接 spawn shell（Rust 端管理） |

参考项目：BongoCat（Tauri 2 + Vue 3 + Vite 结构）

---

## 核心功能

### 一、分屏

- 支持水平拆分和垂直拆分
- 快捷键拆分（可自定义）
- 拖拽调整分屏比例
- 每个分屏独立 PTY 会话
- 支持嵌套拆分（分屏中再分屏）

### 二、全局快捷键唤起

- 注册系统级全局快捷键（默认如 `Ctrl+~` 或 `Cmd+~`）
- 按下后立即唤起/隐藏终端窗口
- 用户可在设置中自定义快捷键
- 支持 Quake 模式（从屏幕顶部下拉）可选

### 三、本地 PTY

- Tauri Rust 后端直接 spawn shell（zsh/bash/powershell）
- 通过 Tauri IPC 将 PTY 输入输出传递给前端 xterm.js
- 自动检测系统默认 shell
- 支持在设置中指定自定义 shell 路径

### 四、多 Tab

- 顶部 Tab 栏，新建/关闭/切换 Tab
- 每个 Tab 可独立分屏
- Tab 标题自动显示当前进程名或路径
- 快捷键切换 Tab（Cmd/Ctrl + 数字）

## 多端同步 & 内嵌 Web 服务

### 核心设计

桌面客户端启动时**内嵌 Axum Web 服务器**（即现有的 Web 服务），手机浏览器通过局域网 IP 访问即可使用完整的 Web 终端功能。

```
┌──────────────────────────────────────────┐
│           桌面客户端 (Tauri)              │
│                                          │
│  ┌────────────┐    ┌─────────────────┐   │
│  │ 本地终端 UI │    │ 内嵌 Axum 服务  │   │
│  │ (Tauri IPC)│    │ (HTTP+WebSocket)│   │
│  └─────┬──────┘    └───────┬─────────┘   │
│        │                   │             │
│        ▼                   ▼             │
│  ┌─────────────────────────────────┐     │
│  │        PTY 会话管理              │     │
│  │   （多端共享同一 PTY 实例）       │     │
│  └─────────────────────────────────┘     │
└──────────────────────────────────────────┘
         ▲                   ▲
         │ Tauri IPC         │ WebSocket
         │                   │
    桌面端操作           手机浏览器访问
```

### 同步规则

- 桌面端和手机端操作**同一个 PTY 会话**，输出实时广播到所有客户端
- 现有 Web 版功能完全保留，不受桌面端影响
- 任一端的输入对其他端实时可见
- Tab / 分屏状态同步（桌面端新建的 Tab 在手机端可见）

### 启动模式

| 模式 | 命令 | 说明 |
|------|------|------|
| 默认模式 | `xterm` | 启动桌面 GUI + 内嵌 Web 服务 |
| 服务端模式 | `xterm --server` | 仅启动 Web 服务，不开启桌面窗口（适合 headless / 远程机器） |

### Web 服务配置

- 默认监听端口：8999（与现有一致）
- 可在设置中修改端口，命令行可通过 `--port` 指定
- 启动时显示局域网访问地址，方便手机扫码或输入

---

## 与手机 Web 版的关系

| 维度 | 手机 Web 版 | 桌面客户端 |
|------|------------|-----------|
| PTY 通信 | WebSocket（连接桌面端） | Tauri IPC（本地直连） |
| 终端渲染 | xterm.js | xterm.js（共用） |
| 键盘 | 虚拟键盘 | 系统键盘 |
| 设置方案 | 共用 | 共用 + 桌面扩展 |
| 分屏 | 简单分屏 | 嵌套分屏 |
| PTY 来源 | 桌面端内嵌服务 | 本地直接 spawn |
